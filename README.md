# IT Helpdesk System

A modern IT helpdesk ticketing system built with React and Vite.

## 🚀 Features

- **User Authentication** - Secure login system
- **Ticket Management** - Create, track, and manage support tickets
- **Dashboard** - Overview of system statistics and activities
- **Settings** - User and system configuration
- **Responsive Design** - Works seamlessly on all devices

## 🛠️ Technologies Used

- **React** - Frontend library
- **Vite** - Build tool and development server
- **React Router DOM** - Client-side routing
- **CSS3** - Styling and animations

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/iMohammedJameel/-IT-Helpdesk-System.git
cd frontend/it-helpdesk
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to:
```
http://localhost:5173
```
